# Ticket 08 — 대화 화면 Immersive Studio 리디자인

  ## 문제 요약
  현재 drama-scene.tsx는 채팅앱처럼 보이며 1057줄의 복잡한 로직과 UI가 혼재.

  ## 목표 레이아웃 (3-zone design)

  ```
  ┌──────────────────────────────────────────────────┐
  │ ZONE A — Scene Header (고정)                     │
  │  [←] 장면 이름  |  Progress bar  |  [설정⚙]    │
  │  "공항 체크인"  ●●●●○○○○○○  10%  [톤: 공손체]   │
  ├──────────────────────────────────────────────────┤
  │ ZONE B — Dialogue Feed (스크롤)                  │
  │                                                  │
  │  ┌─────────────────────────────────────────┐    │
  │  │ [AI 캐릭터] 아바타 + 이름                │    │
  │  │ 「いらっしゃいませ。ご予約はございますか？」│    │
  │  │  [▶] 듣기  |  韓: 어서오세요. 예약 있으...│    │
  │  │  読み: いらっしゃいませ。ごよやく...      │    │
  │  └─────────────────────────────────────────┘    │
  │                                                  │
  │  ┌─────────────────────────────────────────┐    │
  │  │                          [나]           │    │
  │  │              「はい、田中と申します」      │    │
  │  │                                         │    │
  │  │  ┌ Coaching ──────────────────────┐    │    │
  │  │  │ ✨ Natural                      │    │    │
  │  │  │ 더 자연스러운 표현:              │    │    │
  │  │  │ 「田中と申しますが」             │    │    │
  │  │  └────────────────────────────────┘    │    │
  │  └─────────────────────────────────────────┘    │
  ├──────────────────────────────────────────────────┤
  │ ZONE C — Action Bar (고정)                       │
  │  [힌트💡] [다시듣기🔁]  [●마이크]  [번역👁]     │
  └──────────────────────────────────────────────────┘
  ```

  ## 세부 변경 사항

  ### Zone A - Scene Header
  - 씬 이름 텍스트 (시나리오 title)
  - 슬림 progress bar (h-1, amber 색상)
  - 우상단 설정 아이콘 (subtitle 토글, 자동청취 모드 등)

  ### Zone B - Dialogue Feed
  **AI 말풍선:**
  - 왼쪽 정렬, 캐릭터 아바타(이름 이니셜 or imageUrl)
  - 일본어 텍스트 (큰 폰트, 읽기 편하게)
  - 자막바(한국어/발음) — subtitleSettings에 따라 토글
  - 재생 버튼 인라인

  **사용자 말풍선:**
  - 오른쪽 정렬
  - 내 발화 텍스트
  - 바로 아래 Coaching card (접히지 않고 인라인 표시):
    - quality label: "Natural" / "Clear" / "Slightly awkward" / "Try this"
    - correction 문장 (있을 때만)
    - betterExpression (있을 때만)

  ### Zone C - Action Bar
  버튼 구성:
  - 힌트: 현재 장면 표현 힌트 팝업
  - 다시듣기: 최근 AI 발화 재생
  - 마이크 (메인 버튼, 크게): press-to-record
  - 번역 토글: 한국어 자막 on/off

  ### 기존 코드 정리 대상
  ```
  drama-scene.tsx 에서 제거/이동:
  - 과도한 console.log 디버그 출력 (lines 123-127 등)
  - "🔥 강제 번역 텍스트" 같은 임시 디버그 문자열 (line 246-247)
  - system turn에서 "English practice" 하드코딩 (Ticket 05와 중복, 여기서도 정리)
  ```

  ## 완료 기준
  - 3-zone 레이아웃 유지 (header / feed / actionbar)
  - coaching card가 사용자 말풍선 바로 아래 인라인 표시
  - 디버그 console.log 및 임시 문자열 제거
  - 마이크 버튼이 액션바 중앙에 크게 배치
